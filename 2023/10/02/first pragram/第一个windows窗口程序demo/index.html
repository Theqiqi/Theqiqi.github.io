新建一个Windows Desktop空项目步骤

  1. 打开VisualStudio2022
  2. 在新建项目里选择windows Desktop Wizard
  3. 点击Creat按钮
  4. 在Application type选项中选择Desktop Application(.exe)
  5. 在Additional options中勾选Empty project
  6. 点击OK按钮
  7. 在源文件中添加一个文件命名为Demo.cpp
