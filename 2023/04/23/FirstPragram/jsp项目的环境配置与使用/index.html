下载完成后安装。

使用eclipse开发第一个jsp项目

  1. 运行eclipse并配置Tomcat

  2. 新建jsp项目

  3. 新建jsp文件

  4. 运行jsp项目

运行eclipse

  1. 通过双击eclipse运行eclipse->选择workspace路径->之后会进入eclipse主界面.
  2. 在Windows->preferences中选择Server->Runtime Environments.
  3. 点击Add->Apache->Apache Tomcat v10.1,点击next.
  4. 在Tomcat installation directory中填入Tomcat的安装目录->点击Finish.

新建jsp项目

  1. 点击file->new->Dynamic Web Project
  2. 在弹出的界面中输入项目名称hello
  3. 点击next->next->Finish.

新建jsp文件

  1. 打开hello->src->main->webapp
  2. 鼠标右键webapp->选择New->JSP File，
  3. 更改文件名为hello.jsp->选择html5->点击Finish.
  4. 在body标签里输入hello world.

运行jsp项目

  1. 在hello.jsp文件中点击鼠标右键选择Run As->Run on Server
  2. 在弹出的界面中选择Tomcat服务器->点击Next->选择hello项目->点击Finish.
  3. 运行后在浏览器中会显示hello world,运行结果如下图所示:
