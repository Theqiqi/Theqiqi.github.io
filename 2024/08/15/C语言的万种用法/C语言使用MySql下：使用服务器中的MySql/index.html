2. 在kali中开启MySql

  1. 打开MySql服务

  2. 查看MySql状态

  3. MySQL初始化

  4. 初始化之后再重启服务

  5. 运行MySQL测试一下，exit退出

3. 开启服务器中MySql的访问权限，使其他主机可以连接服务器的MySql数据库

  1. 在服务器中启动MySql

  2. 列出所有MySql用户

  3. 选择数据库mysql

  4. 显示数据库mysql,里面保存了所有用户

  5. 创建新用户命名为new

  6. 设置网段内所有主机都可连接用户new

  7. 刷新mysql设置

  8. 退出mysql程序

  1. 创建数据库
  2. 选择数据库
  3. 创建表
  4. 给表中插入数据
  5. 退出

二、编写C语言程序并运行

1.编写C语言程序

  1. 将本地MySql中的include与lib库包含在项目中

  2. 编写C语言代码

  3. 编译

  4. 新建一个文件夹，将应用程序与相关Dll文件添加到文件夹中

  5. 运行文件夹里的应用程序

2.解决程序连接服务器失败的问题

  1. 在本地使用ping命令测试服务器是否连通，不连通的话自行排查网络问题，连通后进行下一步。

  2. 在服务器里使用命令查看端口连通状态

    • 结果显示地址为0.0.0.0:3306说明可被其他地址连接，如果为127.0.0.1:3306则说明MySql设置了只允许本机地址连接.
    • 解决办法为修改相关文件允许所有ip连接

  3. 修改文件

    • 将文件里的bind-address从127.0.0.1修改为0.0.0.0

  4. 重启MySql服务
